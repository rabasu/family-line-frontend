## 概要
### 小岩井農場が誇る「日本三大牝系」のトップ

小岩井農場が1907年に輸入した基礎牝馬の1頭「**フロリースカツプ**」を牝祖とするファミリーライン。

戦前から多くの御賞典馬が輩出し、戦後には**コダマ**、**ニホンピロウイナー**、**スペシャルウィーク**らが活躍。

21世紀に入っても**メイショウサムソン**と**ウオッカ**がダービーを制覇。**レイパパレ**が2021年の大阪杯（GI）を勝利するなど、令和になっても高い活力を維持している。

def fib():
    a, b = 0, 1
    while True:            # First iteration:
        yield a            # yield 0 to start with and then
        a, b = b, a + b    # a will now be 1, and b will also be 1, (0 + 1)

for index, fibonacci_number in zip(range(10), fib()):
     print('{i:3}: {f:3}'.format(i=index, f=fibonacci_number))
